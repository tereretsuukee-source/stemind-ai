# STEMind

> **Triple-verified mentorship. Personalized growth. Built for every learner.**

STEMind connects students with rigorously vetted mentors for 1-on-1 learning sessions across coding, design, career preparation, and beyond. No guesswork. Just expert guidance tailored to your goals.

---

## ✨ Features

- **Triple-Verified Mentors** — Every expert is identity-checked, skill-tested, and peer-reviewed.
- **1-on-1 Live Sessions** — Book personalized video or chat sessions that fit your schedule.
- **Diverse Topics** — From programming and UX design to interview prep and academic coaching.
- **Student-First Pricing** — Flexible plans designed for learners, not corporations.
- **Goal Tracking** — Set milestones and measure progress with your mentor over time.
- **Secure & Private** — End-to-end encrypted sessions and protected personal data.

---

## 🚀 Getting Started

### Prerequisites

- Node.js `>= 18.0.0`
- npm or yarn
- A modern web browser

### Installation

```bash
# Clone the repository
git clone https://github.com/your-org/stemind.git
cd stemind

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:3000`.

## 🛠 Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React / Next.js / Tailwind CSS |
| Backend | Node.js / Express |
